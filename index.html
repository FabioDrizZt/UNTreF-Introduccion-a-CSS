Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML. Permite separar el contenido de la presentación, lo que facilita el mantenimiento y la reutilización del código.

¿Qué es CSS?

CSS es un lenguaje de hojas de estilo que se utiliza para definir la apariencia y el formato de un documento HTML. Permite aplicar estilos a elementos HTML, como colores, fuentes, márgenes y mucho más.
